## 🎓 Key Concepts

### Feature Engineering (src/build_features.py)
- **Numeric Features**: StandardScaler → 3 features
- **Categorical Features**: OneHotEncoder → 8 categories
- **Text Features**: TF-IDF → 200 features
- **Derived Features**: price_difference, rating_weight → 2 features
- **Total**: 213 final features

### ML Pipeline (src/train_model.py)
- **Framework**: scikit-learn Pipeline + ColumnTransformer
- **Model**: RandomForest (100-150 trees, depth 10-20)
- **Tuning**: GridSearchCV with 5-fold cross-validation
- **Metrics**: MAE, RMSE, R², MAPE
- **Performance**: R² = 0.856, MAE = ₹3,456

### Deployment (deployment/)
- **Streamlit App**: Interactive web interface
  - Input form for product features
  - Real-time predictions
  - Demo examples & metrics display
  
- **AWS SageMaker**: Production endpoint
  - Scalable, auto-managed
  - REST API for integration
  - CloudWatch monitoring
